Difference between div and span:-

  1. Div
    • Block-level element
    • Takes up the full width available(starts on a new line)
    • Used to group larger chunks of content, sections, or layouts.
  2. Span
    • Inline element
    • Takes up only as much width as its content, does not start on a new line.
    • Used for styling or grouping small pieces of text within a line.

Difference between class and id:-

  1. Class
    • Can be applied to multiple elements (reusable).
    • Used when you want to style or select a group of elements.
    • In CSS, it is targeted with a . (dot).
  2. Id
    • Must be unique in a page (used for one specific element).
    • Used when you want to style or select a single element.
    • In CSS, it is targeted with a # (hash).

Difference between em and strong:-

  1. em(emphasis)
    • Used to emphasize a word or phrase.
    • Default Style: Usually italic.
    • Semantic Value: Tells browsers, screen readers, and search engines that the text has stress emphasis.
  2. strong
    • Used to indicate strong importance, seriousness, or urgency.
    • Default Style: Usually bold.
    • Semantic Value: Signals that the text is very important.
